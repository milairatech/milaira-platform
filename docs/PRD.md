# Character Counter - Product Requirements Document

## Overview

A free online tool that counts:

* Characters
* Characters (without spaces)
* Words
* Sentences
* Paragraphs

in real time.

## Target Users

* Students
* Writers
* Bloggers
* Social Media Managers
* Developers

## Features

### Core Features

* Real-time counting
* Copy text
* Clear text
* Mobile responsive

### Statistics

* Character count
* Character count without spaces
* Word count
* Sentence count
* Paragraph count

## Future Features

* Keyword density
